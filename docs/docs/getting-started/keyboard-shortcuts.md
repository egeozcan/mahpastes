# Keyboard Shortcuts

Complete reference for all keyboard shortcuts in mahpastes.

## Global Shortcuts

These work anywhere in the application.

| Shortcut | Action |
|----------|--------|
| <span className="keyboard-key">Cmd</span>/<span className="keyboard-key">Ctrl</span> + <span className="keyboard-key">V</span> | Paste from system clipboard |
| <span className="keyboard-key">Esc</span> | Close current modal or dialog |

## Image Editor Shortcuts

When the image editor is open, use these tool shortcuts.

### Tool Selection

| Shortcut | Tool | Description |
|----------|------|-------------|
| <span className="keyboard-key">B</span> | Brush | Freehand drawing |
| <span className="keyboard-key">L</span> | Line | Draw straight lines |
| <span className="keyboard-key">R</span> | Rectangle | Draw rectangles |
| <span className="keyboard-key">C</span> | Circle | Draw circles/ellipses |
| <span className="keyboard-key">T</span> | Text | Add text annotations |
| <span className="keyboard-key">E</span> | Eraser | Erase annotations |

### Editing

| Shortcut | Action |
|----------|--------|
| <span className="keyboard-key">Cmd</span>/<span className="keyboard-key">Ctrl</span> + <span className="keyboard-key">Z</span> | Undo last action |
| <span className="keyboard-key">Cmd</span>/<span className="keyboard-key">Ctrl</span> + <span className="keyboard-key">Y</span> | Redo undone action |
| <span className="keyboard-key">Cmd</span>/<span className="keyboard-key">Ctrl</span> + <span className="keyboard-key">Shift</span> + <span className="keyboard-key">Z</span> | Redo (alternative) |

:::tip Undo History
The image editor maintains up to 50 undo steps. You can undo and redo freely without losing your work.
:::

## Image Comparison Shortcuts

When comparing two images:

| Shortcut | Action |
|----------|--------|
| <span className="keyboard-key">←</span> / <span className="keyboard-key">→</span> | Move slider (in slider mode) |
| <span className="keyboard-key">+</span> / <span className="keyboard-key">-</span> | Zoom in/out |
| <span className="keyboard-key">0</span> | Reset zoom to fit |

## Selection Shortcuts

| Shortcut | Action |
|----------|--------|
| <span className="keyboard-key">Shift</span> + Click | Add/remove clip from selection |
| <span className="keyboard-key">Cmd</span>/<span className="keyboard-key">Ctrl</span> + <span className="keyboard-key">A</span> | Select all visible clips |

## Tips for Efficient Use

### Quick Workflow

1. <span className="keyboard-key">Cmd</span>/<span className="keyboard-key">Ctrl</span> + <span className="keyboard-key">V</span> to paste
2. Click to view/edit
3. <span className="keyboard-key">Esc</span> to close and continue

### Image Annotation Flow

1. Open image editor
2. <span className="keyboard-key">R</span> for rectangle to highlight areas
3. <span className="keyboard-key">T</span> for text to add labels
4. <span className="keyboard-key">Cmd</span>/<span className="keyboard-key">Ctrl</span> + <span className="keyboard-key">Z</span> if you make a mistake
5. Save when done

### Bulk Operations

1. <span className="keyboard-key">Shift</span> + click to select multiple clips
2. Use the action bar for bulk archive/delete/download

## Platform Differences

| Action | macOS | Windows/Linux |
|--------|-------|---------------|
| Paste | <span className="keyboard-key">Cmd</span> + <span className="keyboard-key">V</span> | <span className="keyboard-key">Ctrl</span> + <span className="keyboard-key">V</span> |
| Undo | <span className="keyboard-key">Cmd</span> + <span className="keyboard-key">Z</span> | <span className="keyboard-key">Ctrl</span> + <span className="keyboard-key">Z</span> |
| Redo | <span className="keyboard-key">Cmd</span> + <span className="keyboard-key">Y</span> | <span className="keyboard-key">Ctrl</span> + <span className="keyboard-key">Y</span> |
| Select All | <span className="keyboard-key">Cmd</span> + <span className="keyboard-key">A</span> | <span className="keyboard-key">Ctrl</span> + <span className="keyboard-key">A</span> |

## Customization

Currently, keyboard shortcuts cannot be customized. This feature may be added in a future release.

If custom shortcuts are important to your workflow, please [open an issue](https://github.com/egeozcan/mahpastes/issues) on GitHub.
